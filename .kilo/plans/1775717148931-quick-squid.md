# Plan: Modern Clock & Weather Display

## Overview

Transform the current placeholder React component into a modern, minimal clock and weather display for MagicMirror². Uses Open-Meteo API (free, no key required) for weather data. Digital clock with date + current weather conditions.

## Architecture

- **No new dependencies** — fetch API is built-in
- **No node_helper** — Open-Meteo is free & keyless, direct browser fetch is fine
- Config-driven: latitude/longitude/timezone/weatherRefreshInterval in MagicMirror config.

## Open-Meteo API Details

- Endpoint: `https://api.open-meteo.com/v1/forecast`
- Params: `latitude`, `longitude`, `current=temperature_2m,weather_code,wind_speed_10m,is_day`, `timezone=auto`
- Free, no API key required
- Response includes `current` object with the requested variables

## WMO Code → Condition Mapping

| Code Range | Condition                | Symbol |
| ---------- | ------------------------ | ------ |
| 0          | Clear sky                | ☀️/🌙  |
| 1-3        | Partly cloudy / Overcast | ⛅     |
| 45, 48     | Fog                      | 🌫️     |
| 51-57      | Drizzle                  | 🌦️     |
| 61-67      | Rain                     | 🌧️     |
| 71-77      | Snow                     | 🌨️     |
| 80-82      | Rain showers             | 🌦️     |
| 85-86      | Snow showers             | 🌨️     |
| 95-99      | Thunderstorm             | ⛈️     |

(is_day field determines day/night icon variant)

## Component Structure

```
App
├── ClockDisplay
│   ├── Time (HH:MM:SS, large)
│   └── Date (weekday, month day, year)
└── WeatherDisplay
    ├── Temperature (e.g. "18°")
    ├── Condition icon + text (e.g. "⛅ Partly Cloudy")
    └── Wind speed (e.g. "12 km/h")
```

All in a single `App.tsx` file with inline sub-components (no separate files needed for this scale).

## Visual Design

- Dark background, white text (MagicMirror standard)
- Time: ~72px, font-weight 200, letter-spacing 0.15em
- Date: ~18px, font-weight 300, letter-spacing 0.1em, opacity 0.7
- Weather: ~24px, font-weight 300, with separator line between clock and weather
- Font: Inter (Google Fonts) with system sans-serif fallback
- Transitions: opacity fade-in for weather data
