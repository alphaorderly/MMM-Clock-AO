# Plan: Modernize CalendarDisplay with Square-ish Minimal Design

## Goal

Redesign `CalendarDisplay.tsx` to look more modern using square elements with minimal rounding (not circular, not sharp-square — subtle `rounded-sm` or `rounded`).

## Changes

### 1. Today highlight: `rounded-full` → `rounded`

- Change today's date from a circle (`rounded-full`) to a subtle rounded square (`rounded`)
- Keep the `bg-white text-black` contrast but make it feel more structured

### 2. Remove circle constraint on date cells

- Change `w-7 h-7` to `w-full h-8` so dates fill their cell as compact square blocks
- This gives a more grid-like, modern feel vs floating circles

### 3. Adjust date cell styling

- Today: `rounded` background fill, `bg-white text-black font-medium`
- Other days: keep colored text, no background shape
- Add subtle `hover:opacity-100` feel by defaulting to slightly higher opacity

### 4. Add weekday row divider

- Add a thin horizontal line under day labels (`border-b border-white/10`) to visually separate header from grid

### 5. Month header refinement

- Slightly larger text (`text-sm`), add `font-medium` for hierarchy
- Use `opacity-60` instead of `opacity-70` for subtlety

### 6. Grid spacing

- Use `gap-0.5` for both x and y gaps for a tighter, more structured grid
- Each date cell: square-ish with `rounded` on today highlight only

## Summary of className changes

| Element                  | Before                                                   | After                                                        |
| ------------------------ | -------------------------------------------------------- | ------------------------------------------------------------ |
| Month header             | `text-[13px] font-light tracking-widest opacity-70 mb-3` | `text-sm font-medium tracking-wider opacity-60 mb-2`         |
| Day labels               | `text-[11px] font-light py-1`                            | `text-[11px] font-medium py-1 uppercase tracking-wide`       |
| Date grid                | `grid grid-cols-7 w-full gap-y-0.5`                      | `grid grid-cols-7 w-full gap-0.5`                            |
| Day labels row container | `grid grid-cols-7 w-full mb-1`                           | `grid grid-cols-7 w-full mb-1 pb-1 border-b border-white/10` |
| Today date span          | `rounded-full w-7 h-7`                                   | `rounded w-full aspect-square`                               |
| Other date span          | `w-7 h-7 rounded-full`                                   | `w-full aspect-square rounded`                               |

No circular elements will remain — all shapes use `rounded` (2px) or `rounded-sm` (1px) for a minimal square aesthetic.
